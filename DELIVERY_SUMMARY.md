## 🌐 DEPLOY TO HEROKU (Live Website)

### Prerequisites
- Download Git: https://git-scm.com/download/win
- Download Heroku CLI: https://devcenter.heroku.com/articles/heroku-cli
- Create Heroku account: https://www.heroku.com (FREE)

### Deploy (5 minutes)
```powershell
# Initialize git
git init
git add .
git commit -m "Initial commit"

# Login
heroku login

# Create app
heroku create your-unique-app-name

# Configure
heroku config:set MONGODB_URI="your-mongodb-connection"
heroku config:set EMAIL_PASS="aqtc ldjx oecp lzuu"

# Deploy
git push heroku main

# Open
heroku open
```

**Your website is now LIVE!** 🌍

After deployment, point `zaki.com` to your Heroku app:

## 🚨 TROUBLESHOOTING

### Port 5000 in use?
```powershell
netstat -ano | findstr :5000
taskkill /PID [number] /F
```

### MongoDB not connecting?
```powershell
# Start MongoDB (Windows)
mongod

# Or use cloud MongoDB
# Update MONGODB_URI in .env
```

### Emails not sending?
- Verify 2FA enabled on Gmail
- Check App Password is correct
- Check `.env` has correct credentials

### Heroku deployment fails?
```powershell
heroku logs --tail
```
